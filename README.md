# 特徴
- シンプルな計算機システム
- キーボード入力に対応

# インストール
- zipファイルをダウンロード
- ダウンロードしたzipファイルを展開
- `rnl-calculator`を`resource`フォルダに導入
- `server.cfg`に`ensure rnl-calculator`を追加
- サーバーを再起動

# 使用方法
`/cal`コマンドを使用して計算機UIを表示<br>
![rnl-calclator-screenshot](rnl-calclator-screenshot.png)
- 数字ボタンを押して値を入力
- ACボタンを押して入力をリセット
- ←ボタンを押して値を1つずつ削除
- 右の記号を押して計算
